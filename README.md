# Flag Football Playbook

Interactive play visualization and game day coaching tool for 5v5 youth flag football.

## Features
- 35 animated play diagrams with route visualization
- Double-Back formation suite (10 plays) — counter, boot, hook-ladder, I-bone, speed pitch, split sweep, counter pass, dual wheel, screen
- Game day call sheet with situation-based recommendations
- Play tracking with success/fail stats
- Roster management and lineup builder
- Wristband play sheet generator (printable)
- Play editor with canvas route drawing
- Sunlight mode for outdoor readability
- PWA — installable, works offline

## Usage
Open `index.html` in any browser, or visit the [live app](https://aigeninc.github.io/ffb-pb/).
